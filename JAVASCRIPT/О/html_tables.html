Табличные теги

table - главный тег таблицы. Он определяет таблицу и содержит в себе все ее теги.

tr - тег строки таблицы. Он определяет строку таблицы и содержит в себе все ее теги.

td - тег ячейки таблицы. Он определяет ячейку таблицы и содержит в себе все ее теги.

Эммет сокращение для быстрого создания таблицы с lorem в ячейках table>tr*2>td*2>lorem

Группа селекторов child

Селекторы child позволяют выбрать элементы, которые являются прямыми потомками других элементов.

Селектор child записывается через >

Селектор child выбирает только прямых потомков, а не всех потомков.

Селектор child выбирает только элементы, которые являются непосредственными потомками указанного элемента.

Селектор child выбирает только элементы, которые находятся на один уровень вложенности с указанным
