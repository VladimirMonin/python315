Строки и их методы

Строки в JavaScript - это последовательность символов, используемых для представления текста. Строки заключаются в одинарные или двойные кавычки.

Характеристики строк:

  • Строки в JavaScript - это последовательность символов, используемых для представления текста.
  • Строки заключаются в одинарные или двойные кавычки.
  • Строки - это примитивные значения и не могут быть изменены.
  • Строки могут быть объявлены с помощью одинарных, двойных или обратных кавычек.
  • Строки могут быть записаны как однострочные, так и многострочные.

Функция typeof

Оператор typeof возвращает строку, указывающую тип операнда.

Синтаксис:

typeof operand

Функция проверки длины строки length

Свойство length возвращает длину строки.

Синтаксис:

string.length

Массивы

Array - это объект, который используется для хранения множества значений в одной переменной. Массивы могут содержать множество элементов, которые могут быть доступны по индексу.

Характеристики массивов:

  • Массивы в JavaScript - это объекты, которые используются для хранения множества значений в одной переменной.
  • Массивы могут содержать множество элементов, которые могут быть доступны по индексу.
  • Массивы могут быть созданы с помощью литерала массива или с помощью конструктора массива.
  • Массивы могут содержать элементы разных типов данных.
  • Массивы могут быть многомерными.
  • Массивы могут быть изменены.

Примеры синтаксиса массивов:


            let array = [1, 2, 3, 4, 5];
            let array = new Array(1, 2, 3, 4, 5);
            let array = new Array(5);
Метод Описание
push() Добавляет один или несколько элементов в конец массива и возвращает новую длину массива.
pop() Удаляет последний элемент из массива и возвращает его.

map() Создает новый массив с результатами вызова предоставленной функции для каждого элемента массива.
filter() Создает новый массив со всеми элементами, которые проходят тест, предоставляемый функцией.
find() Возвращает первый элемент массива, который удовлетворяет предоставленной функции проверки.
findIndex() Возвращает индекс первого элемента массива, который удовлетворяет предоставленной функции проверки.
reduce() Применяет функцию к аккумулятору и каждому элементу массива (слева направо), чтобы свести его к одному значению.
splice() Изменяет содержимое массива, удаляя, заменяя или добавляя элементы.
slice() Возвращает поверхностную копию части массива в новый объект массива.
join() Соединяет все элементы массива в строку.
concat() Используется для соединения двух или более массивов.
reverse() Обращает порядок элементов массива на месте.
sort() Сортирует элементы массива на месте и возвращает отсортированный массив.
includes() Определяет, содержит ли массив определенный элемент, и возвращает true или false.
indexOf() Возвращает первый индекс, по которому данный элемент может быть найден в массиве, или -1, если его нет.
some() Проверяет, соответствует ли хотя бы один элемент массива предоставленной функции проверки.
every() Проверяет, соответствуют ли все элементы массива предоставленной функции проверки.
fill() Заполняет все элементы массива от начального до конечного индексов одним значением.
from() Создает новый экземпляр массива из итерируемого объекта.
isArray() Определяет, является ли переданный аргумент массивом.
flat() Создает новый массив с элементами подмассивов в одной последовательности.
flatMap() По сути комбинирует методы map и flat в одной операции.

Объекты в JS

Объекты в JavaScript - это контейнеры для именованных значений, называемых свойствами и методами.

Характеристики объектов:

  • Объекты в JavaScript - это контейнеры для именованных значений, называемых свойствами и методами.
  • Объекты могут быть созданы с помощью литерала объекта или с помощью конструктора объекта.
  • Объекты могут содержать множество свойств и методов.
  • Свойства и методы объекта могут быть примитивными значениями, объектами или функциями.
  • Объекты могут быть многомерными.
  • Объекты могут быть изменены.

Примеры синтаксиса объектов:


            let object = {
                name: 'John',
                age: 30,
                hobbies: ['music', 'movies', 'sports'],
                address: {
                    street: '50 Main st',
                    city: 'Boston',
                    state: 'MA'
                }
            };

Создание нового объекта:


            let object = new Object();
            let object = Object.create(null);
